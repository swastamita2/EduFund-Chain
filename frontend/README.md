# EduFund Frontend
Bagian ini berisi kode sumber antarmuka pengguna untuk proyek EduFund Chain.

## Status Pengembangan
Saat ini pengembangan frontend berada pada tahap perancangan komponen dasar. Implementasi logika fungsional dan integrasi Web3 dijadwalkan untuk diselesaikan pada tahap UAS.

## Organisasi File
* **src/components/**: Direktori untuk komponen UI React yang dapat digunakan kembali.
* **src/components/DonationCard.jsx**: Komponen utama untuk interaksi donasi.

## Catatan Teknis
Aplikasi ini menggunakan React.js dan akan diintegrasikan dengan MetaMask menggunakan library Ethers.js.
