# EduFund Chain
Sistem Donasi Transparan untuk Institusi Pendidikan Usia Dini Berbasis Ethereum

## Deskripsi Proyek
EduFund Chain adalah platform berbasis blockchain yang dirancang untuk meningkatkan transparansi dan akuntabilitas dalam pengelolaan donasi pada institusi pendidikan usia dini (PAUD/TK). Dengan memanfaatkan teknologi Smart Contract di jaringan Ethereum, setiap transaksi donasi dicatat secara permanen dan dapat diverifikasi oleh publik melalui block explorer.

## Struktur Repository
Penyusunan folder dalam repository ini mengikuti standar manajemen proyek perangkat lunak untuk memastikan keberlanjutan pengembangan hingga tahap UAS.

* **docs/**: Berisi dokumen laporan formal (Format Word dan Markdown) serta aset diagram sistem.
* **contracts/**: Berisi kode sumber Smart Contract yang ditulis menggunakan bahasa Solidity.
* **frontend/**: Berisi kode sumber antarmuka pengguna berbasis React.js (Tahap Pengembangan).
* **project-charter/**: Berisi dokumen manajemen tim, pembagian peran, dan jadwal kerja.
